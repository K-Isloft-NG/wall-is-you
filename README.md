# Wall Is You

Jeu de labyrinthe en Python développé dans le cadre de la SAÉ 1.01 
du BUT Informatique à l'IUT de Marne-la-Vallée.

## Présentation

Le joueur doit trouver la sortie d'un labyrinthe généré algorithmiquement. 
Le chemin est calculé via un algorithme BFS (Breadth-First Search).

## Fonctionnalités

- 3 niveaux de difficulté (facile, moyen, difficile)
- Mode aléatoire — labyrinthe généré à chaque partie
- Interface graphique via Pygame
- Architecture modulaire (moteur, UI, personnages)

## Stack technique

- Python 3
- Pygame (interface graphique)
- BFS — Breadth-First Search (résolution de labyrinthe)

## Lancer le projet

```bash
pip install pygame
python main.py
```
